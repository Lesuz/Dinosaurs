Dinosaurs

What is this app?

This app uses a collection of data I have put into a database. The data inludes a name of a dinosaur species with some information about them, or better said, what a friend of mine thinks of them.

How to navigate this app?

To get all the information in the API, add "/api/getall" to the url. It will show all the data in a json format on the screen. To get back to these instructions, come back to this page.

To get information on a specific dinosaur, add "/api/dinosaurname" to the url. Or type a word like saur to get all the dinosaurs with "saur" in their name.
